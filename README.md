# Tdraw - 终端绘图工具

一个基于Rust TUI的简单终端绘图程序，允许你在终端中绘制和操作图形区块。

## 功能特点

- **自适应画布**：自动获取当前终端大小，并创建白色画布
- **圆点导航**：使用 `wasd` 键控制圆点（模拟鼠标）在画布上移动
- **区块创建**：按住 `Shift` 键后，使用方向键（上下左右）确定区块的大小，松开 `Shift` 键确认区块的对角位置
- **区块交互**：圆点移动到区块上时，区块边框颜色会发生变化
- **区块选择**：按 `Enter` 键选中区块，之后可以使用 `wasd` 键带着选中的区块一起移动

## 使用说明

1. 启动程序后，终端会显示一个白色画布
2. 使用 `wasd` 键移动圆点
3. 按住 `Shift` 键并配合方向键创建区块
4. 移动圆点到区块上查看高亮效果
5. 按 `Enter` 选中区块，然后使用 `wasd` 键移动选中的区块
